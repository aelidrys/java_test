# Contract API

Mini projet de test technique Spring Boot - Gestion des contrats.

## Lancer le projet

```bash
mvn spring-boot:run
```

Accès à la console H2 : http://localhost:8080/h2-console  
Documentation OpenAPI : http://localhost:8080/swagger-ui.html

---
## Runing the server

- #### mkdir postgres_db
- #### run docker compose up -d
- #### cd sboot_crud
- #### mvn spring-boot:run
